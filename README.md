## Project Overview
This project focuses on designing chatbot personas using prompt engineering techniques.
Each chatbot has a defined tone, behavior, and conversational flow for real-world use cases.

---

## Chatbot Personas

### 1. AI Mental Health Companion
- Tone: Calm, empathetic, supportive
- Use Case: Stress relief, daily check-ins
- Features:
  - Emotional validation
  - Gentle coping suggestions
  - Safe fallback responses

### 2. AI Subject Tutor
- Tone: Supportive, informative
- Use Case: Explaining academic topics
- Features:
  - Step-by-step explanations
  - Simple examples
  - Quiz-style interaction

### 3. AI Travel Guide
- Tone: Energetic, informative
- Use Case: Travel planning and tips
- Features:
  - Budget suggestions
  - Packing tips
  - Destination recommendations
